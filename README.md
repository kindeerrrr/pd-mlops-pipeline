# PD MLOps Pipeline

Проект по автоматизации полного MLOps-пайплайна для модели
предсказания вероятности дефолта клиента (PD-модель).

## Цель проекта
Реализовать воспроизводимый и автоматизированный pipeline:
- подготовка данных
- обучение модели
- валидация и тестирование
- логирование экспериментов
- развёртывание API

## Датасет
Default of Credit Card Clients Dataset (UCI ML Repository)

## Структура проекта
- src/ — основной код проекта
- tests/ — unit-тесты
- notebooks/ — исследования и EDA
- data/ — данные
- models/ — сохранённые модели
